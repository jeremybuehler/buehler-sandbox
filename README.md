## Deploy
### Vercel
- Import the repo/folder
- Framework: **Vite**
- Build: `npm run build`
- Output directory: `dist`
- `vercel.json` is included.

### Netlify
- Create new site from Git or drag & drop the project
- Build command: `npm run build`
- Publish directory: `dist`
- `netlify.toml` is included.
